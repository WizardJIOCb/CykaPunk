# Documentation

This folder contains all server configuration files and deployment documentation for the CykaPunk project.

## Files Included:

- **nginx-game.cyka.lol.conf** - HTTP Nginx configuration for the domain
- **nginx-game.cyka.lol-ssl.conf** - HTTPS Nginx configuration with SSL support
- **DEPLOYMENT.md** - Complete deployment guide and server setup instructions

## Quick Start:

1. Copy the nginx configuration to your server
2. Follow the deployment guide in DEPLOYMENT.md
3. Set up SSL certificate with Certbot for production use
